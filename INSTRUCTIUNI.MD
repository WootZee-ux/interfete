Fiecare student va realiza o aplicație software, utilizând una dintre tehnologiile Java Swing, JavaFX, Python sau Visual C#. Aplicația trebuie să includă cel puțin șase ecrane grafice (ferestre/scene), interconectate prin meniuri, butoane sau alte elemente grafice de navigare. Tematica proiectului este la alegere, însă aplicația trebuie să urmeze un flux logic — de exemplu un ecran de autentificare, urmat de un ecran principal, module de gestionare a datelor, un formular cu un test grilă, un ecran de afișare a informațiilor, un ecran de help etc. Fiecare fereastră va afișa în bara de titlu numele studentului și grupa. Fiecare dintre aceste ecrane trebuie să conțină elemente grafice precum butoane, etichete, câmpuri de introducere a datelor, liste, tabele, meniuri etc. Codul sursă al proiectului trebuie să fie structurat în clase, organizat corespunzător și comentat.

Se recomandă personalizarea interfețelor (culori, fonturi, iconițe).
